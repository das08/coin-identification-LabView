# coin-identification-LabView

## 開発環境等
- LabView 2020 
- myDAQ device

## 概要
1円玉から500円玉まで６種類の硬貨を自動識別する装置のソフトウェア部分です。

LabViewを使うことでDAQ APIを簡単に利用でき、myDAQでの測定が容易になります。

本ソフトウェアは、
- Function Generator
- Oscillo & Auto Identification
の2つによって構成されています。
